# Crypto Time Series Forecasting (BTC & ETH) using FEDformer

Цей репозиторій містить реалізацію прогнозування часових рядів
на основі архітектури **FEDformer** для двох криптовалют:
**Bitcoin (BTC)** та **Ethereum (ETH)**.

Проєкт створений у рамках дослідження для дипломної роботи та включає:
- повну реалізацію моделі FEDformer,
- підготовку даних,
- експерименти з прогнозуванням BTC та ETH.
